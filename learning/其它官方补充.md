
### 单文件组件  
> 使用 `.vue` 的单文件组件，可以解决全局定义的问题、实现高亮语法、预处理器等。  
> 
> 在一个组件里，其模板、逻辑和样式是内部耦合的，这样搭配使得组件更加内聚且更可维护。

```
// 组件作用域的 CSS
<style scoped>
...
</style>
```
